# Health On Wheels - Site Web

Un site web moderne et professionnel pour le projet Health On Wheels, une initiative à impact social portée par l'équipe MACAS qui fournit des services de santé mobiles aux communautés rurales à travers le monde.

## 🎯 Objectif

Créer une présence en ligne professionnelle pour Health On Wheels afin de :
- Présenter le projet et sa mission
- Mobiliser des partenaires et des soutiens financiers
- Informer sur l'impact social de l'initiative
- Faciliter les contacts et les collaborations

## 🏗️ Structure du Projet

```
MACAS/
├── index.html          # Page principale avec toutes les sections
├── styles.css          # Feuille de style complète avec thème clair/sombre
├── script.js           # Interactions JavaScript et animations
└── README.md           # Documentation du projet
```

### 1. Page d'Accueil (Hero)

### 3. Parties Prenantes

- Message inspirant

- Impact mis en avant

## 🎨 Caractéristiques Design

### Thème et Couleurs
- **Palette santé** : Blanc, orange (#ff6b35), gris foncé
- **Mode clair/sombre** avec transition fluide
- **Contrastes optimisés** pour l'accessibilité

### Responsive Design
- **Mobile-first** approach
- **Breakpoints** : 768px (tablettes), 480px (mobiles)
- **Navigation adaptative** avec menu hamburger

### Animations et Interactions
- **Scroll fluide** entre sections

- **Compteurs animés** pour les statistiques
- **Barre de progression** de lecture
